# wsp-socket-io-client

wsp-socket-io-client es un ejemplo para escuchar eventos de un servidor que tiene una conexi√≥n con socket.io y una instancia de whatsapp-web.js y asi poder comunicarse con whatsapp desde un cliente web y viceversa.

## Installation

Use the package manager [pnpm](https://pnpm.io/es/installation) or [npm](https://nodejs.org/en) to install wsp-socket-io-client.

```bash
# Para instalar todas las dependencias del servidor
# con pnpm
pnpm install
# con npm
npm install
# inicia el servidor
pnpm run dev
# o
npm run dev

# Para instalar todas las dependencias del cliente
cd frontend
# con pnpm
pnpm install
# con npm
npm install
# inicia el cliente
pnpm run dev
# o
npm run dev
```

## Usage

```bash
# Para hacer un build del cliente
cd frontend
# con pnpm
pnpm run build
# o
npm run build
```
